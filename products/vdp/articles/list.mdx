## Unified Vulnerability Repository

SnapSec's Unified Vulnerability Repository functions as a centralized storage hub, consolidating all reported vulnerabilities from diverse sources, including different assessments and automation tools like SnapSec ASM. 

This centralization provides a unified platform for comprehensive analysis and an efficient response to security issues, offering an eagle-eye view of all vulnerabilities reported to this vulnerability Management System, regardless of their origin.

**List of table columns in the vulnerability list page:**

| Property        | Description                                         |
| --------------- | --------------------------------------------------- |
| **#ID**             | Unique identifier for the reported vulnerability.  |
| **REPORTED ON**     | Date when the vulnerability was reported.            |
| **TITLE**           | Brief title or description of the reported issue.   |
| **REPORTED BY**     | Email of the security researcher who reported the issue. |
| **STATUS**          | Current state of the reported vulnerability (e.g., open, resolved, in progress). |
| **SEVERITY**        | Level of severity assigned to the reported vulnerability (e.g., low, medium, high). |
| **LAST ACTIVITY**   | Date of the last activity or update related to the reported vulnerability. |


## Status and Severity Tracking

SnapSec's tracking system monitors the real-time status (open, resolved, in progress) and severity levels of each vulnerability. This dynamic tracking empowers users to prioritize their efforts based on the urgency and potential impact of each identified security weakness.

## Filtering


### Status based Filtering
By simply clicking on a specific vulnerability state in the vulnerability list will filter and show only vulnerabilities that fall into the selected state. This feature allows you to narrow down and focus on vulnerabilities based on their current status, facilitating a more detailed examination and management of security issues in the chosen state.
